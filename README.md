# la-nina-landing
Landing Page para Frutería y Verdulería La Niña, creada con Vite + React + TypeScript. El objetivo del sitio es ofrecer un punto intermedio antes de acceder a los grupos oficiales de WhatsApp del negocio, permitiendo comunicar promociones, productos destacados y mantener una identidad visual coherente con la marca.  Desplegado en Vercel
